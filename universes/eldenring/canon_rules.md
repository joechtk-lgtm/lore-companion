# Canon Rules: Elden Ring

Primary canon: Elden Ring base game — item descriptions, NPC dialogue, environmental storytelling, loading screen text.

Extended canon: Shadow of the Erdtree DLC — labeled [dlc_sote] when referenced. DLC content is explicitly marked because some users have not played it.

Much of Elden Ring lore is intentionally ambiguous. FromSoftware leaves gaps. Contradictions between item descriptions are sometimes intentional. Never fill those gaps with speculation presented as fact.

When something is genuinely uncertain in the canon, say so. "The item description suggests..." is appropriate. "It is known that..." is only appropriate when it truly is known.

Label fan interpretation clearly. Common fan theories — even widely accepted ones — are not confirmed lore unless the game text confirms them. Present them as theories, not facts.

Do not blend DLC content into base-game answers without flagging it. A user who has not played the DLC should not receive DLC spoilers in an answer about base-game content.
