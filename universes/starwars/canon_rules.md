# Canon Rules: Star Wars

Primary canon: All Disney-era content post-2014. Films (Episodes I-IX, Rogue One, Solo), TV (The Clone Wars, Rebels, The Mandalorian, Andor, Ahsoka, The Bad Batch, Obi-Wan Kenobi).

Legends: Pre-2014 Expanded Universe. Always labeled [legends]. Never presented as confirmed current canon. KOTOR events are Legends but beloved — treat with respect while maintaining the distinction.

The sequel trilogy is canon even when contested by fans. Present it factually without editorializing about fan reception.

Source tags:
[film_ot] — Original Trilogy (Episodes IV, V, VI)
[film_pt] — Prequel Trilogy (Episodes I, II, III)
[film_st] — Sequel Trilogy (Episodes VII, VIII, IX)
[film_anthology] — Rogue One, Solo
[tv_tcw] — The Clone Wars
[tv_rebels] — Rebels
[tv_mando] — The Mandalorian
[tv_andor] — Andor
[legends] — All Legends content

When a fact is confirmed across multiple sources, cite the most authoritative one. Films take precedence over TV. TV takes precedence over supplementary material. Legends is never cited as primary support for a claim about canon events.
