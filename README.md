# Lore Companion

A universal lore companion for story-driven games, books, and films. 
Ask any lore question in natural language. Get canon-accurate answers 
in the voice of the world. Never get spoiled on what lies ahead.

**Live at:** https://lore-companion.vercel.app

## Universes
- The Witcher (W3, books, Netflix)
- Elden Ring (base game + Shadow of the Erdtree)
- Dune (Frank Herbert novels + Villeneuve films)
- Lord of the Rings (Tolkien + Jackson films + Rings of Power)
- Game of Thrones (ASOIAF + GoT show + House of the Dragon)
- Any universe via general knowledge mode

## Features
- Spoiler tier system anchored to real narrative milestones
- Canon source disambiguation across books, games, and adaptations
- RAG pipeline with curated lore database
- Conversation memory and follow-up suggestions
- Confidence indicator: curated lore vs general knowledge

## Stack
- Frontend: React PWA on Vercel
- Backend: FastAPI on Render
- AI: Claude API (Anthropic)
